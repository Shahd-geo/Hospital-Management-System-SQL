# Hospital Management System - SQL Server Implementation

This project contains the SQL Server implementation for a Hospital Management System database.

## Description

The database was implemented based on the ERD and Mapping diagrams using SQL Server and T-SQL.

## Features

- Primary Keys (PK)
- Foreign Keys (FK)
- Composite Primary Keys
- One-to-Many Relationships
- Many-to-Many Relationships
- Self-contained relational structure
- CHECK Constraints
- Multi-valued Attribute Mapping

## Database Tables

- Patient
- Patient_Phone
- Doctor
- Doctor_Phone
- Department
- Appointment
- Services
- Appointment_Services
- Medical_Record
- Billing

## Concepts Applied

- ERD to Relational Mapping
- Associative Entities
- Derived Attributes
- Data Integrity Constraints
- Referential Integrity

## Technologies

- SQL Server
- T-SQL

The project includes complete CREATE TABLE statements with constraints and relationships.
